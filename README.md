# E-Commerce-Sales-Analysis-Dashboard-Power-BI
Power BI E-Commerce Sales Analysis Dashboard built to analyze revenue, AOV, orders, cancellations, and lost revenue. Used Power Query for data cleaning, DAX for KPIs, and interactive visuals to track performance by product, category, state, and time for business decision-making.

What I Did in This Project

Cleaned and transformed raw data using Power Query

Created relationships between Orders, Products, and Customers

Built DAX measures:

Total Revenue

Average Order Value (AOV)

Total Orders

Cancellation Rate

Lost Revenue due to Cancellations

Designed an interactive dashboard with:

Category slicer

Date range filter

Product, category, state, and time-based analysis

Applied dashboard layout, colors, and formatting

Locked visuals to prevent accidental movement
